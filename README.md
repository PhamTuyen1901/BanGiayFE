# BanGiayFE

Frontend project for BanGiay application built with Next.js

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- React

## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Run the development server:
```bash
npm run dev
```

## Features

- Modern UI/UX design
- Responsive layout
- Fast performance
- Type-safe development with TypeScript
